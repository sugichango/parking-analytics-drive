■ 🔑 ダッシュボードV2用 Googleドライブ連携設定マニュアル

Googleドライブに置いたデータをプログラムに読み込ませるための「合鍵（サービスアカウント）」を作る手順です。
※難しそうに見えますが、画面の通りに進めるだけで数分で終わります。

--------------------------------------------------
手順1. Google Cloudでプロジェクトを作る
--------------------------------------------------
1. 普段お使いのGoogleアカウントにログインした状態で、以下のURLを開きます。
   https://console.cloud.google.com/
2. 左上の「プロジェクトの選択」をクリックし、「新しいプロジェクト」を作成します。
   （名前は何でも構いません。例：「parking-dashboard」など）

--------------------------------------------------
手順2. Google Drive API を有効にする
--------------------------------------------------
1. 左上の「≡ (メニュー)」から「APIとサービス」＞「有効なAPIとサービス」を開きます。
2. 上の「＋ APIとサービスの有効化」をクリックします。
3. 検索窓に「Google Drive API」と入力して検索し、クリックします。
4. 青い「有効にする」ボタンをクリックします。

--------------------------------------------------
手順3. 合鍵（サービスアカウント）を発行する
--------------------------------------------------
1. 左のメニューから「APIとサービス」＞「認証情報」をクリックします。
2. 上の「＋ 認証情報を作成」をクリックし、「サービス アカウント」を選びます。
3. サービスアカウント名を入力します。（例：「dashboard-reader」など）
4. 「作成して続行」を押し、ロール（役割）は「Basic」の「閲覧者」などを選び、「完了」を押します。
5. 作成されたサービスアカウントの「メールアドレス（長い英数字@...iam.gserviceaccount.com）」を**コピーしてメモしておきます**。

--------------------------------------------------
手順4. 合鍵の実体（JSONファイル）をダウンロードする
--------------------------------------------------
1. 作成したサービスアカウントの名前をクリックします。
2. 上の「キー」タブを開き、「鍵を追加」＞「新しい鍵を作成」を選びます。
3. キーのタイプは「JSON」が選ばれた状態で「作成」を押します。
4. パソコンにファイル（秘密の鍵）がダウンロードされます。
   ※この中身のテキストは、後ほどStreamlitのSecretsに貼り付けます。

--------------------------------------------------
手順5. Googleドライブでフォルダを共有する
--------------------------------------------------
1. ご自身のGoogleドライブを開き、ダッシュボード用のデータを入れる専用フォルダを作ります。
   （例：「DashboardData」という名前のフォルダ）
2. そのフォルダを右クリックして「共有」を選びます。
3. 共有相手の追加欄に、**【手順3-5】でメモした長いメールアドレス**を貼り付けて追加します。
4. 権限は「閲覧者」または「編集者」にして保存します。
   ※これで、この長いメールアドレスを持つプログラムだけが、このフォルダの中身を覗けるようになります。他の人には一切見えません。

--------------------------------------------------
これで準備完了です！
この手順でダウンロードしたJSONファイルの中身を、StreamlitのSecrets画面に後ほど設定します。
