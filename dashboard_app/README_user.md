# 🅿️ 駐車場データダッシュボード 使い方マニュアル

本システムは、駐車場の利用データをインタラクティブに分析するための「統合ダッシュボード」です。クラウド（インターネット）を通じて、いつでも最新のデータを閲覧・分析できます。

## 1. アプリの起動方法

本システムは以下のURLをクリックするだけで、ブラウザから簡単にアクセス可能です。

### 🌐 アクセス用URL
[**駐車場データダッシュボードを開く**](https://parking-yo789img2hvubb5jqzexzgd.streamlit.app)

> **💡 ヒント**：
> お使いのブラウザ（Microsoft EdgeやGoogle Chrome）の「お気に入り」や「ブックマーク」に登録しておくと、次回からワンクリックで開けて便利です。PCだけでなく、タブレットやスマートフォンからも閲覧可能です。

---

## 2. ログインの方法
URLを開くと「ログイン画面」が表示されます。

1. **メールアドレス欄**：
   ご自身のドメイン（例: `name@tutc.or.jp`）を入力してください。
2. **パスワード欄**：
   管理者から支給された共通ログインパスワードを入力してください。
3. **「ログイン」ボタン** をクリックすると、分析画面が表示されます。

---

## 3. ダッシュボードの構成
画面左側のサイドバーメニューから、分析モードを切り替えられます。

### ① 一般利用台数推移分析
* **目的**：駐車場の収入と利用台数の「月別・日別」の全体推移を把握します。
* **ポイント**：左側のフィルターで「特定の駐車場」や「平日のみ」などに絞り込めます。「支払い種別表示」にチェックを入れると、現金・回数券などの内訳が色分けされます。

### ② 24時間稼働状況分析
* **目的**：時間帯ごとの混雑状況（波形）や、定期券利用者と一般利用客の比率を詳しく分析します。
* **ポイント**：上部のタブ（1〜5）を切り替えて視点を変えられます。数字の横にある **「？」マーク** にマウスを合わせると、その項目の見方や判断基準の解説が表示されます。

---

## トラブルシューティング
* **画面が動かなくなった**：ブラウザを更新（F5キー）してください。
* **「許可されていないドメインです」と出る**：メールアドレスの末尾が `@tutc.or.jp` になっているか確認してください。最後に余計な「スペース」が入っていないかも注意してください。
* **パスワードを忘れた**：システム管理者に確認してください。
* **ログアウトしたい**：左サイドバーの一番下にある「ログアウト」ボタンを押してください。
