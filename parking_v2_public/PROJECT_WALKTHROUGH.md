# 【完全版】駐車場分析統合ダッシュボード：構築からクラウド移行、そして知見の継承まで

本プロジェクトは、ローカル環境での高度なデータ分析から始まり、Google Drive を活用したクラウド共有システム、そしてその全ノウハウをAIスキルとして保存するまでを完遂しました。

---

## 🪐 プロジェクトの変遷

### フェーズ 1：ローカル・アナリティクスの確立 (V1)
- **課題**: 散らばったCSV/Excelデータから駐車場利用の実態を素早く可視化すること。
- **成果**: 
    - 複雑な割引コードから「RB/回数券/現金」を正確に分類する判定ロジックを開発。
    - 平日・休日を分離した24時間稼働波形の生成アルゴリズムを確立。
    - Mode ① (台数推移) と Mode ② (稼働状況) の二本柱からなるダッシュボードを構築。

### フェーズ 2：Google Drive 連携とクラウド移行 (V2)
- **課題**: 職員がどこからでも、最新のドライブ上のデータを確認できるようにすること。
- **成果**:
    - Google Drive API を組み込み、共有フォルダからの自動データ同期を実現。
    - Streamlit Cloud の制約（Pythonバージョン、依存関係）をクリアし、本番環境を安定化。

### フェーズ 3：究極の視認性とノウハウの永続化
- **課題**: 公開後の利便性を高め、将来的な保守を容易にすること。
- **成果**:
    - 暗い背景でもKPIが完璧に読める「極限可読CSS」の実装。
    - Antigravity および ClaudeCode の両方がこのプロジェクトを理解できるよう **「Skillファイル」** および **「CLAUDE.md」** を作成。
