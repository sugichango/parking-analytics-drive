# 🅿️ 駐車場統合アナリティクス (Google Drive 連携版) 運用マニュアル

このシステムは、Google ドライブ上の駐車場利用データを自動的に読み込み、Streamlit Cloud 上で高度な可視化・分析を行うためのツールです。

## 📂 2. データの更新方法（管理者向け）

Google ドライブの共有フォルダに以下のファイルを保存・更新することで、ダッシュボードに反映されます。

### ① 日次・月次利用台数の更新
- **ファイル名**: `updated_integrated_data_FY2025.csv.gz`
- **内容**: 一般利用客の入出庫・現金収入データ。
- **更新頻度**: 月に一度、または必要に応じて最新版を上書き保存してください。

### ② 24時間稼働状況（時間帯別平均）の更新
- **ファイル名形式**: `[年度]_[駐車場名]_with_avg.xlsx` (例: `2025_南1駐車場_with_avg.xlsx`)
- **内容**: 時間帯別の入出庫台数、在庫台数の平均データ。
- **注意点**: ファイル名に「年度」と「駐車場名」が含まれていることを確認してください。

---

## 📊 3. 主要機能の解説

### メニュー ①：一般利用台数推移分析
- 駐車場ごとの利用台数と、現金収入の推移を二軸グラフで表示します。
- **RB・回数券・現金** の内訳をチェックボックス一つで切り替え可能です。

### メニュー ②：24時間稼働状況分析
- 時間帯別のピーク時刻や、**定期依存度（月極利用の割合）** を算出します。
- 平日と休日の稼働ギャップを確認し、満車対策や収益改善の検討に活用できます。

## 🛠️ 4. トラブルシューティング

### 画面が「Oh no.」と表示されて起動しない
- **原因**: サーバーの Python バージョン設定の不整合が多いです。
- **対策**: Streamlit Cloud の Settings で Python バージョンが `3.12` になっているか確認してください。

### ログインしても「ファイルが見つかりません」と出る
- **原因**: Google ドライブ上のファイル名がマニュアルと異なっている可能性があります。
- **対策**: ファイル名が `updated_integrated_data_FY2025.csv.gz` 等と完全に一致しているか確認してください。
