# 駐車場データダッシュボード 使い方マニュアル

本システムは、駐車場の利用データをインタラクティブに分析するための「統合ダッシュボード」です。
インターネットに繋がっていなくても、PC上で直接ブラウザを開いて動作させることができます。

## 1. アプリの起動方法

### デスクトップから起動する場合
デスクトップにある **「駐車場データダッシュボード起動」** のアイコンをダブルクリックしてください。

### フォルダから直接起動する場合
同封されている `launch_dashboard.bat` というファイルをダブルクリックしてください。

> **初回起動時の注意点**
> 初回起動時のみ、裏側でPythonの環境構築（必要なエンジンのインストール）が数分間行われます。黒い画面が表示されたままになりますが、そのまま待機してください。完了すると自動的にブラウザが立ち上がります。

## 2. ログインの方法
ブラウザが立ち上がると「ログイン画面」が表示されます。

1. **メールアドレス欄**：
   ご自身のドメイン（例: `hoge@tutc.or.jp`）を入力してください。※許可されていないメールアドレスの場合は弾かれます。
2. **パスワード欄**：
   支給された共通ログインパスワードを入れてください。
3. **「ログイン」ボタン** をクリックすると、分析画面に入れます。

## 3. ダッシュボードの構成

画面左側のメニューから、見たいモードを切り替えることができます。

### ① 一般利用台数推移分析 (旧：経営ダッシュボード)
* **主な機能**：駐車場の収入と利用台数の「月別・日別」の全体推移を確認できます。
* **使い方**：
  - 左サイドバーの「フィルター設定」から、特定の駐車場や月、平日/休日を絞り込むことができます。
  - 「支払い種別表示オプション」にチェックを入れると、現金・回数券・回数券併用などの内訳がグラフに色付きで表示されます。

### ② 24時間稼働状況分析 (旧：稼動分析プロ)
* **主な機能**：24時間の中での「波形（ピークや空き時間）」や「月極（定期）と一般の比率」を深く掘り下げる分析です。
* **使い方**：
  - 上部の「タブ（1〜5）」を切り替えることで、分析の切り口（平日と休日の比較や、定期・一般の割合など）を簡単に変えられます。
  - 数値が並んでいるカード（KPIカード）の「？」アイコンにマウスを合わせると、その数値の意味や計算式、目安となる解説（ポテンシャル判断基準など）が表示されます。

## トラブルシューティング
* **画面が真っ白になる / グラフが出ない**
  画面を更新（F5キー）してみてください。
* **「データファイルが見つかりません」と表示される**
  `data` フォルダの中身が移動されたか、消えてしまった可能性があります。システム管理者に連絡してください。
* **終了したい場合**
  ブラウザのタブを閉じた後、起動時に裏側で立ち上がっていた「黒い画面（コマンドプロンプト）」も「×」で閉じてください。
